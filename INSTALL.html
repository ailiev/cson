Cson Installation

Prerequisites

  • A C++ compiler.
  • GHC haskell compiler.
  • bnfc version 2.3b or later, to compile the Cson parsing and pretty-printing library.
  • The flex lexer generator, and the bison parser generator.
  • For the Haskell target, need alex lexer generator, happy parser generator, and ghc compiler. See the Faerieplay compiler install notes for information about these.

Building

$ cd BASEDIR/json
$ make install
  • Builds the C++ component and installs the shared libraries.
  • Builds the Haskell component, which can be used directly from this build directory, by adding BASEDIR/json/bnfc/ to the Haskell library path.
