C-Json Installation

Prerequisites

  • A C++ compiler.
  • GHC haskell compiler.
  • bnfc version 2.3b or later, to compile the C-Json parsing and pretty-printing library.
  • The flex lexer generator, and the bison parser generator.
  • For the Haskell target, need alex lexer generator, and happy parser generator.

Building

$ make bnfc
$ make install
Builds and installs the C++ component. Just builds the Haskell component, which can be used directly from this build directory, by adding json/bnfc/ to the Haskell library path.
